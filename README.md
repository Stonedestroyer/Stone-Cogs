# Stone-Cogs
Random Cogs for Red Bot V3.

# Installation

Replace [p] with your prefix and [cogname] with the cog you wanna install.

1. [p]load Downloader
2. [p]repo add Stone-Cogs https://github.com/Stonedestroyer/Stone-Cogs.git
3. [p]cog install Stone-Cogs [cogname]
